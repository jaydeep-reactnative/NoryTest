# NoryTest

##  Project Desctiption
Nory is a demo application with responsive UI implemented using Chakra UI and also used Redux and React Router for state management and navigation. Implemented API and handled data seamlessly using Redux. 

### `yarn install`

Install the packages mentioned in the package.json<br />

### `yarn start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br />
You will also see any lint errors in the console.

## Main technologies used

- [React](https://reactjs.org/)
is a JavaScript library for building user interfaces.

- [Redux](https://redux.js.org/)
is a predictable state container for JavaScript apps. .

- [React Router](https://reactrouter.com/)
is a collection of navigational components that compose declaratively with application. 

- [Chakra UI ](https://chakra-ui.com/)
is a simple, modular and accessible component library that gives you the building blocks you need to build your React applications.
